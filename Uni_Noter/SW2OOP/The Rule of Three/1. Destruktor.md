
Deklaration 
`````c++
class MyClass {
public:
    MyClass(int size);
    ~MyClass();          // destruktor
private:
    int* _data;
    int  _size;
};
`````

Implementering 
`````c++
MyClass::MyClass(int size)
    : _size(size), _data(new int[size]) {}

MyClass::~MyClass() {
    delete[] _data;   // frigiv array
    _data = nullptr; // god vane
}
`````

Main 
`````c++
{
    MyClass a(10); // new int[10]
} // <-- destruktor kaldes her automatisk
// delete[] sker — ingen memory leak
`````
